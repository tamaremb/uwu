# ¡Hola, Mundo! en C 🌟

Este es un proyecto extremadamente sencillo en lenguaje **C** que imprime el clásico mensaje de saludo `"¡Hola, mundo!"` en la terminal. Sirve como plantilla base o punto de partida para configurar y probar entornos de desarrollo en C.

## 📋 Requisitos Previos

Para compilar y ejecutar este programa, necesitarás un compilador de C instalado en tu sistema.

- **macOS**: Xcode Command Line Tools (`clang`) o `gcc` instalado mediante Homebrew.
- **Linux**: GCC (usualmente disponible a través de `sudo apt install build-essential`).
- **Windows**: MinGW-w64, MSYS2 o Visual Studio.

## 🚀 Compilación y Ejecución

Puedes compilar y ejecutar el proyecto manualmente desde tu terminal.

### 1. Compilar el archivo

Ejecuta el siguiente comando en la raíz del proyecto:

```bash
gcc hola.c -o hola
```

*Nota: Esto generará un archivo ejecutable llamado `hola` (o `hola.exe` en Windows).*

### 2. Ejecutar el programa

Una vez compilado, puedes correr el ejecutable con:

```bash
./hola
```

Deberías ver la siguiente salida en tu pantalla:
```text
¡Hola, mundo!
```

## 🛠️ Configuración de VS Code

El proyecto incluye una carpeta `.vscode` con configuraciones recomendadas para compilar y depurar en Visual Studio Code de manera sencilla si tienes la extensión de C/C++ instalada.
